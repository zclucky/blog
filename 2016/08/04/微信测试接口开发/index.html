新浪云平台创建一个空的php应用


选择使用git作为代码托管工具

  • 本地工作空间地址打开git命令行
  • git init初始化
  • git remote add sae 仓库地址
  • 本地文件夹创建一个index.php文件
  • git add .添加到暂存库
  • git commit -am 'add a new file'提交到服务器
  • git push sae master:1 提交到git库主分支 版本1
  • 刷新新浪云平台控制器 查看代码是否提交到云上


###配置微信测试账号接口信息

  • URL中填写新浪云平台项目生成的地址


  • TOKEN 自定义口令
  • 下载微信测试接口提供的示例php文件 http://mp.weixin.qq.com/mpres/htmledition/res/wx_sample.zip
  • 将微信测试接口提供的示例php文件copy到index.php文件中去
  • 测试配置是否成功
  • 进入https://mp.weixin.qq.com/debug/开始测试相关接口
