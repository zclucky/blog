mysql服务常用命令

我们在使用mysql服务的时候会用到一些命令，比如启动服务，停止服务啊等。

  • net start mysql57，启动mysql服务，后面的mysql57是上面我们安装步骤里面的服务名。

环境变量

安装完毕之后，如果你没改安装路径的话，默认安装路径应该是C:\Program Files\MySQL\MySQL Server 5.7\，如果我们要在命令行cmd里执行mysql的一些指令的话，就会用C:\Program Files\MySQL\MySQL Server 5.7\bin\mysql.exe这个程序，但是你没有必要每次都输入这么长一串字符，这里的话我们就需要用到环境变量。
我们把C:\Program Files\MySQL\MySQL Server 5.7\bin\这个目录添加到环境变量里面去，然后每次只需要输入mysql，系统会自动到的C:\Program Files\MySQL\MySQL Server 5.7\bin\目录去找到mysql.exe执行程序。
如果你是以前版本的mysql的话，手动设置：

  1. 右键点击此电脑=>属性=>高级系统设置=>环境变量=>系统变量=>path
  2. 如果你是windows10的话，点击新建，然后把C:\Program Files\MySQL\MySQL Server 5.7\bin\地址添加进去。如果是windows7的话，就直接在后面加上英文小写的;，后面把路径添加进去即可。
  3. 一路点击确定，完成环境变量的配置。

这样通过打开windows下的命令行cmd，就可以看到mysql输出：

修改编码

因为我们都是使用的操作系统是中文简体操作系统，所以数据库里面客户端编码是gbk，

这会导致后面很多中文字体出现乱码的情况，为了使我们后面的学习没有后顾之忧，必须修改所有编码为通用编码utf8，这就需要修改mysql的配置文件my.ini，首先找到上面的数据安装目录C:\ProgramData\MySQL\MySQL Server 5.7，找到里面的my.ini文件，复制一份到桌面上来。然后通过sublime text等编辑工具打开它，只需要修改下面三点：

  • [client]，找到这个下面，添加default-character-set=utf8。
  • [mysql]，找到这个下面，添加default-character-set=utf8。
  • [mysqld]，找到这个下面，添加character-set-server=utf8。

然后把修改后的my.ini文件粘贴到mysql的安装目录C:\Program Files\MySQL\MySQL Server 5.7\去，最后，重启mysql服务再进去就OK了。

自此，windows10下面安装和配置mysql就全部完毕，谢谢。
